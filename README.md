# Manutenção V5 Premium

App em Streamlit com visual premium estilo MES/Tesla para controle de manutenção.

## Recursos
- Login por perfil
- Abertura de OS corretiva
- Painel com iniciar, pausar, retomar e finalizar
- Tempo de resposta, reparo e parada
- Cadastro de máquinas, técnicos, peças e usuários
- Baixa automática de peças
- Custos de mão de obra e peças
- Upload de foto da falha
- Assinatura digitada do operador e técnico
- Plano preventivo
- Dashboard premium com KPIs

## Como rodar
```bash
pip install -r requirements.txt
streamlit run app.py
```
