# Manutenção V12.1 Industrial

Esta versão une:
- compatibilidade com o banco da V10
- manutenção dos cadastros de máquinas e técnicos
- preventiva com geração automática de OS
- painel industrial estilo Tesla/MES

## Como manter os cadastros da V10
Use esta versão no mesmo ambiente do arquivo:
`manutencao_v10_fabrica.db`

## Rodar
pip install -r requirements.txt
streamlit run app.py
